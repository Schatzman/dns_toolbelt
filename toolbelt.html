DNS Troubleshooting Toolbelt

DNS Checker Tool Aggregate

Dig Web Interface - Domain Record Checker

Useful for the ability to query various nameservers including authoritative to quickly check if records exist even when not propagated.


The default search is for the host name's A records against all available nameservers. Keep in mind that within the frame it is possible to search for multiple host names at once with this tool as well as checking other types of records and only with specific nameservers.

http://www.digwebinterface.com/

MX Toolbox - Domain Record Checker

Useful for checking that MX records and especially spf records have been created properly.


The default search is for the domain's mx records. Remember that this tool can check other domain records as well, such as spf and srv.

http://mxtoolbox.com/

WHO.IS - Public Domain Database

Useful for checking the status and ownership of a domain.


Remember that with some registrars such as godaddy, you will need to get the company's internal who.is link from the bottom of the who.is page to see any useful information.

http://who.is/

MYIP.MS - Hosting Info, Websites and IP Database

Useful for determining the owner of an ip address or the ip address of a website.

http://myip.ms/

IS IT DOWN RIGHT NOW?

Check if a website is reachable.

http://www.isitdownrightnow.com/

Domain Tools

Great for seeing if a domain was previously registered, if and how many times the nameservers have been changed. Useful in determining if other content was previously hosted at this domain.


Example: User is seeing unsafe content warning but just connected the site. Might check to see if the domain was connected elsewhere recently, then use the way back machine (http://archive.org/web/) to see what the website used to be and why it might have been flagged as malicious.

http://whois.domaintools.com/

Into DNS

This tool checks many aspects of a domain and often returns useful errors that can offer insight into the nature of DNS issues.


http://www.intodns.com/

Wix.com Domain Troubleshooter

Our own domain connection checker, often provides useful results. It's a flash app so you will need to enter the domain manually.



12 DNS Propogation Checkers:

Since there are many many servers around the world, when unsure if changes to a domain's DNS are still propagating, it is important to check multiple sources. The following are from http://www.quantawebdesign.com/domain-names/12-places-to-check-dns-propagation.html

1. Site24x7
- Checks 43 worldwide locations - you can choose to query all, or just some of them.

2. WhatsMyDNS
- Queries 20 servers worldwide.

3. ViewDNS
- Checks 19 servers worldwide.

4. Ceipam
- Checks 18 servers worldwide, with options to check a custom IP, and your own local DNS server.

5. CheckDNSpropagation
- Checks a number of worldwide servers.

6. Preshweb
- Offers a number of options for checking DNS records. A little unreliable; some servers return no information.

7. Mob
- Checks 19 servers, 15 of which are in the U.S.

8. Nexcess
- Looks up your DNS in 22 global locations.

9. Just-Ping
- Checks 50 locations throughout the globe.

10. DNSUnlimited
- Checks 13 servers worldwide.

11. PropagatedYet
- fill in the domain and new IP and it'll give you a "yes" or "no" answer to the question, "Propagated Yet?"

12. Just DNS Lookup
- Checks 29 locations, worldwide.
